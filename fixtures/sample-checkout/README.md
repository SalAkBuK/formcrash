# Sample checkout fixture contract

The bundled sample uses deterministic, fake-only values:

- Product `resilience-mug`: Resilience Mug, USD 18.00.
- Product `retry-notebook`: Retry Notebook, USD 12.50.
- Test examples use `Ava Example`, `ava@example.test`, and `42 Test Lane`.

The executable catalogue remains owned by `apps/sample-checkout`; this directory
documents values that future saved-journey fixtures must reuse. No real customer,
address, payment, credential, or government-identifier data belongs here.
